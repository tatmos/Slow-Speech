# LoopMaker

Web上で動作するループ波形エディタです。波形ファイルをアップロードし、ループクロスフェード区間を設定して、シームレスなループ音声を作成できます。

## 機能

- **波形アップロード**: 音声ファイル（WAV、MP3など）をアップロード
- **2トラック表示**: 
  - トラック1: フェードイン部分（ループクロスフェード区間）
  - トラック2: フェードアウト部分（ループ位置以降）
  - ステレオ波形の場合は、各トラックに2行で表示
- **ループ位置設定**: スライダーでループクロスフェード区間開始位置を調整（初期値: 75%）
- **プレビュー再生**: 2トラックを同時にループ再生して確認
- **ファイル保存**: 加工された波形をWAVファイルとして保存

## 使い方

1. 「波形をアップロード」ボタンから音声ファイルを選択
2. スライダーでループクロスフェード区間開始位置を調整
3. 「再生」ボタンでプレビューを確認
4. 「ファイル保存」ボタンで加工された波形を保存

## 技術仕様

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Audio API

## ブラウザ要件

- Chrome、Firefox、Edge、Safariなどのモダンブラウザ
- Web Audio APIをサポートしている必要があります
